Statistics Calculator!


Write the web app to prompt accepting for positive integers until the user enters a negative integer


  1. If the entered text is not a positive integer number, then the web app will keep asking for a number until the valid positive integer is entered
  2. If the entered text is a negative integer, then the web app will show the statistics (average, min, max) of all entered positive integers
  3. If the user enters a negative integer as the first integer, the average, the min, and the max is 0

Hint:


  1. Implement these two functions
    list = readInput();
    displayStats(list);
  2. Use ternary operator (conditional operator ?) to handle in both cases which the list is empty and is not empty
