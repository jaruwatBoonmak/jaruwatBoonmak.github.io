Prime Number Finder!


Write the web app to prompt accepting for a positive integer number


  1. If the entered text is not a positive integer number, then the web app will keep asking for a number until the valid positive integer is entered
  2. If the entered text is a positive integer, then the app will print prime numbers that are not greater than that given positive integer

Hint:


  1. you may want to use loop with label
  2. you may want to use array with push function
